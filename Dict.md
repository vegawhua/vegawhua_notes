## Symbol

| Name  | Chinese | Symbol |
| ------------- | ------------- | ------------- |
| parentheses / brackets | 括号  | `()` |
| square brackets  | 方括号  | `[]` |
| curly braces  | 大括号  | `{}` |
| dot  | 句号  | `.` |
| comma  | 逗号  | `,` |
| colon  | 冒号  | `:` |
| exclamation mark | 感叹号 | `!` |
| asterisk  | 星号  | `*` |
| equal sign  | 等号  | `=` |
| dash | 短横线 | `-` | 
| apostrophe  | 撇号  | I'm |
| quotation (marks)  | 单引号  | `'` |
| double quotes | 双引号  | `"` |
| (forward)slash  | 斜杠  | `/` |
| back slash*  | 反斜杠  | `\` |
| angular brackets | 大于小于号 | `<` `>` |
